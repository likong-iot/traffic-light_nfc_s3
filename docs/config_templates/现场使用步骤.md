# 现场使用步骤

## 1. 准备 SD 卡配置

1. 打开本目录中的 `CONFIG.JSN`。
2. 根据现场需求修改时间、NFC 映射或雷达参数。
3. 把修改后的 `CONFIG.JSN` 放到 SD 卡根目录。

SD 卡根目录示例：

```text
/CONFIG.JSN
```

## 2. 上电读取配置

设备启动时读取配置的优先级：

1. SD 卡 `/sdcard/CONFIG.JSN`
2. ESP32 NVS 中保存的配置
3. 固件内置默认配置

如果 SD 卡中有有效配置，设备会使用 SD 卡配置，并同步一份到 NVS。

## 3. 网页修改配置

1. 手机或电脑连接设备 SoftAP。
2. 默认密码：`12345678`。
3. 浏览器打开：`http://192.168.4.1/`。
4. 在网页左侧选择需要修改的分区：时间配置、NFC 映射、雷达输入、系统状态。
5. 点击“保存全部配置”。

保存时设备会同时尝试写入：

- SD 卡 `/sdcard/CONFIG.JSN`
- ESP32 NVS

没有 SD 卡时，网页保存仍会写入 NVS。

## 4. 常见问题排查

### SD 卡配置没有生效

检查以下内容：

- 文件名是否为 `CONFIG.JSN`。
- 文件是否放在 SD 卡根目录。
- JSON 格式是否正确，逗号、引号、括号是否完整。
- SD 卡是否被设备成功挂载。

### NFC 卡触发动作不对

检查以下内容：

- `nfc.rules` 中的 `data` 是否和卡内 `data[0..1]` 一致。
- `opto12_pulses` 是否设置为期望脉冲数。
- 需要断开继电器 4 的规则，`open_relay4` 是否为 `true`。

### 雷达重复触发

检查以下内容：

- `cycle_window_ms` 是否太短。默认是 `20000` ms。
- `active_level` 是否为 `1`。新版 PCB 雷达 INT 为高电平有效。
- 雷达输出是否有抖动或持续高电平。
