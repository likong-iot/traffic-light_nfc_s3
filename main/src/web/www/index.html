AP配置

AP 名称最长 32 字节，密码长度 8~63 字节。保存后写入配置文件和 NVS；当前连接可能仍保持旧 AP，重启后一定使用新配置。

只匹配卡内 data[0..1]，不匹配 UID。00..05 固定对应 1..6 类卡；名称、OPTO1+OPTO2 脉冲数和相关动作时长在同一行配置。
